## About This Repository
This repository contains all my **Artificial Intelligence Laboratory/Project** work for the CSE 3202 course. I implemented various AI algorithms, solved real-world problems using AI techniques, and documented my projects as part of the course requirements.  

I primarily used **Python**, and experimented with **PROLOG** and **LISP** concepts when needed. The work here covers state space search, constraint satisfaction problems, optimization, adversarial search, and hands-on AI projects.
